# 클린코드 스터디

* 교재: ["클린코드, 이제는 파이썬이다"]( https://m.yes24.com/Goods/Detail/111408257 )
* 참여자: @JayFreemandev 혼자

이 스터디의 특징은 다음과 같다.

* 책의 모든 코드를 직접 입력해본다.
* 경우에 따라 책의 코드를 그대로 사용하지 않고 약간의 변형을 가한다.
* 단계별로 git commit 을 하여, 책을 읽으면서 순서대로 따라올 수 있도록 한다.
